# Logbook

Logbook is a project designed to provide an intuitive way to record, manage, and review logs or
notes related to your activities, experiments, or daily work. Built primarily in Python with a web
interface using HTML and CSS, Logbook makes it easy to keep detailed records and access them
from anywhere.

## Features

- **Simple and clean interface:** Log and review your entries with ease.

## Getting Started

### Prerequisites

To run Logbook locally, you need:

- [Python 3.8+](https://www.python.org/downloads/)
- [pip](https://pip.pypa.io/en/stable/installation/)

### Installation

1. **Clone the repository:**

   ```bash
   git clone https://github.com/carvalhocaio/logbook.git
   cd logbook
   ```

2. **Install the dependencies:**

   ```bash
   pip install -r requirements.txt
   ```

### Running the Application

Start the local server with:

```bash
python manage.py runserver
```

or

```bash
make run
```

By default, the application will be accessible at [http://127.0.0.1:8000](http://127.0.0.1:8000).

### Usage

- Access the web interface in your browser.
- Use the menu to add new logs, search existing ones, and organize your entries.
- All logs are saved locally in the specified database/file.
